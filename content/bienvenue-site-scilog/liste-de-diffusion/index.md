## Liste de diffusion générale

Une nouvelle liste de diffusion (en remplacement de gdr.gpl@imag.fr) a été créée : 
annonces.scilog@groupes.renater.fr.

Cette liste a vocation de regrouper toutes les personnes intéressées par les Sciences du Logiciel. L'abonnement est ouvert à tous. Son contenu est modéré mais devrait accepter tout message lié aux thématiques scientifiques du GDR et à son organisation. Typiquement il s'agit de :

  * Annonce d'événements dans la thématique du {{% scilog %}}
  * Annonce d'emplois disponibles

La liste est hébergée par RENATER. Pour vous inscrire :

<https://listes.univ-grenoble-alpes.fr/sympa/subscribe/gdr.gpl?previous_action=info>

:warning: Seuls les abonnés ont le droit de poster sur la liste de diffusion. Assurez-vous que votre adresse d'abonné est bien identique à celle qui apparaît dans le champ "from" quand vous postez un message.


## Listes de diffusions internes

Pour les inscrits à {{% mygdr %}}, il existe un certains nombre de listes :

* gdr.scilog@groupes.renater.fr pour joindre tous les membres du {{% scilog %}} (uniquement des annonces en lien avec le GDR, donc pas d'offres d'emploi par exemple, cf. liste ci-dessus)
* direction.scilog@groupes.renater.fr pour joindre les 3 co-animateurs du {{% scilog %}}
* defi-xyz-gdr-scilog@groupes.renater.fr pour joindre personnes inscrites au défi "xyz" 
* gt-xyz-gdr-scilog@groupes.renater.fr pour joindre personnes inscrites au groupe thématique  "xyz" 

:white_check_mark: Rappel: nous vous invitons vivement à vous inscrire sur {{% mygdr %}} !
